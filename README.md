## Overview
This is a custom Alexa skill designed to entertain users by telling them random jokes. Developed as one of my first major coding projects at age 13, this skill introduced me to voice-based applications and the Amazon Alexa ecosystem.

## Features
- Responds to user voice commands with a random joke.
- Uses Amazon Alexa Skills Kit (ASK) and AWS Lambda.
- Simple and engaging user experience with humorous content.

## Technologies Used
- **Amazon Alexa Skills Kit (ASK)**
- **AWS Lambda (Node.js or Python)**
- **Amazon Developer Console**
- **JSON for intent schema and sample utterances**
